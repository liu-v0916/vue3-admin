## 文件资源目录 📚

```text
Geeker-Admin
├─ .vscode                # vscode推荐配置
├─ mock                   # mock文件
├─ public                 # 静态资源文件（忽略打包）
├─ src
│  ├─ assets              # 静态资源文件
│  ├─ components          # 全局组件
│  ├─ layout              # 框架布局
│  ├─ plugins             # 第三方插件
│  ├─ router              # 路由管理
│  ├─ views               # 项目所有页面
│  ├─ App.vue             # 入口页面
│  └─ main.ts             # 入口文件
├─ vite                   # vite 配置
├─ .env                   # 环境
├─ .gitignore             # git 提交忽略
├─ index.html             # 入口 html
├─ package.json           # 依赖包管理
├─ postcss.config.js      # postcss 配置
├─ README.md              # README 介绍
├─ tailwind.config.cjs    # tailwindcss 配置
├─ tsconfig.json          # typescript 全局配置
└─ vite.config.ts         # vite 配置
├─ yarn-lock.json         # 依赖包包版本锁
```
